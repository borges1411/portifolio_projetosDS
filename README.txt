Portfólio de Projetos de Ciência de Dados
Este é o meu portfólio online de projetos de Ciência de Dados, onde demonstro minhas habilidades e experiências utilizando dados públicos, ferramentas de análise de dados e machine learning. Aqui você encontrará projetos completos, além de informações sobre meu histórico profissional, habilidades técnicas e áreas de especialização.

Visão Geral do Projeto
Este portfólio foi criado utilizando HTML e CSS para fornecer uma interface de fácil navegação, apresentando os principais projetos que desenvolvi ao longo da minha carreira em Ciência de Dados.

Funcionalidades:
Apresentação de projetos de ciência de dados, incluindo descrição dos problemas de negócios e as soluções desenvolvidas.
Seção "Sobre Mim" detalhando minha trajetória profissional e habilidades técnicas.
Lista de habilidades e áreas de especialização, como Análise de Dados, Machine Learning e Visualização de Dados.
Links para redes sociais e contato direto.
Estrutura do Projeto
assets/: Diretório contendo arquivos de estilo (CSS) e imagens.
index.html: Página principal do portfólio.
scripts/: Diretório contendo os arquivos de scripts JavaScript utilizados na página.
Projetos Destacados
Construção de um Programa de Fidelidade com Clusterização de Clientes
Utilizando Python e machine learning não supervisionado, desenvolvi um modelo para segmentar clientes e construir um programa de fidelidade que poderia gerar $15MM em receita adicional para a empresa.

Ferramentas Utilizadas: Python, Pandas, Matplotlib, AWS EC2, S3, K-Means, DBScan.
Identificação de Imóveis para Compra e Revenda
Projeto que visa identificar imóveis abaixo do preço médio de mercado para revenda, utilizando análise de dados e machine learning.

Ferramentas Utilizadas: Python, Seaborn, Plotly, Streamlit, Heroku Cloud.
Tecnologias Utilizadas
HTML5 e CSS3 para a estrutura e design da página.
Python e bibliotecas como Pandas, NumPy, e Scikit-learn para desenvolvimento dos projetos de ciência de dados.
Power BI, Seaborn, Matplotlib e Plotly para visualização de dados.
AWS, Heroku, e Google Cloud para deploy de soluções.
